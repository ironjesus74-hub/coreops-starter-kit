Why CoreOps

Built Different

Runs Anywhere

Android (Termux), Debian, Ubuntu, Alpine — same CLI, same commands. No Docker, no cloud dependency.

Zero Bloat

Pure POSIX shell — no runtimes, no heavy packages. Installs in seconds, responds in milliseconds.

Self-Building

The 4-bot factory auto-generates 24 networking and utility tools — no manual setup needed.

Fully Open

MIT licensed. Read it, audit it, fork it. No subscriptions, no telemetry, no lock-in of any kind.

Quick Start

Termux / Linux Setup

Install in seconds
pkg update -y && pkg install -y git git clone https://github.com/ironjesus74-hub/coreops-starter-kit.git cd coreops-starter-kit chmod +x bin/* modules/*.sh lib/*.sh bots/*.sh install.sh ./install.sh coreops help
CLI Commands

Command Reference

coreops doctor
System health check — verify all dependencies are present.
coreops netcheck
DNS + HTTP connectivity test with pass/fail reporting.
coreops sslcheck <host>
Inspect TLS certificate: expiry, issuer, SANs.
coreops portscan <host>
TCP port range scan — find open ports quickly.
coreops audit <host>
Full HTTP/TLS/DNS audit of a target host.
coreops scan <host>
Quick security scan combining port + TLS checks.
coreops live <host>
Live monitor — continuous polling with refresh interval.
coreops snapshot
Capture system state: IPs, open ports, running services.
coreops deps
Check and install missing tool dependencies.
coreops panel
Interactive GUI control panel for the bot factory.
coreops bots start
Launch all 4 automation bots via the supervisor.
coreops bots status
Show running/stopped status of all bots.
Bot Factory

4-Bot Automation System

Bot Role Output
Supervisor Launches and polices the other 3 bots. Pauses rogue bots. Admin
Builder Bot 1 Builds networking tools: HTTP testers, DNS, port scanners. networking/
Builder Bot 2 Builds wrapper scripts: curl-smart, git-quick, json-formatter. wrappers/
Watchdog Silently monitors all bots, auto-restarts on crash. background
