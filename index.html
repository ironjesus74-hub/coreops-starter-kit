Core Modules

Doctor
System health check — validates environment, shell dependencies, and tool availability.
coreops doctor
🌐
Net Check
Connectivity, latency, and DNS resolution tests in one shot.
coreops netcheck
🔍
Port Scan
Light-weight local port scanner — identify open services without nmap.
coreops portscan
🔐
SSL Check
TLS certificate validation, expiry warnings, and cipher inspection.
coreops sslcheck
📸
Snapshot
Capture full system state — env, processes, network, disk — as a timestamped report.
coreops snapshot
🔎
Security Scan
Surface common misconfigurations and world-writable paths in seconds.
coreops scan

Deps Audit
Check installed packages and version hygiene across your environment.
coreops deps

Quick Start

Quick Start (Termux / Linux)
Install & Run
# Termux bootstrap
pkg update -y && pkg install -y git

# Clone the repo
git clone https://github.com/ironjesus74-hub/coreops-starter-kit.git
cd coreops-starter-kit

# Make executable & install
chmod +x bin/* modules/*.sh lib/*.sh install.sh
./install.sh

# Run a check
coreops doctor
