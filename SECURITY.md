# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| `main`  | ✅ Active          |
| Older   | ❌ No support      |

## Reporting a Vulnerability

**Do not open a public issue for security vulnerabilities.**

Email: ironjesus74@gmail.com

Include:
- Repository and branch/version
- Steps to reproduce
- Expected vs. actual behavior
- Potential impact assessment

## Response Targets

| Stage        | Target         |
| ------------ | -------------- |
| Acknowledge  | 72 hours       |
| Triage       | 7 days         |
| Fix / advise | Best-effort    |

## Scope

This policy covers:
- `coreops-starter-kit` source code (shell scripts, Cloudflare Worker, HTML/JS assets)
- GitHub Actions workflows in `.github/workflows/`

Out of scope:
- Automated tools built by the bot factory at runtime (`~/Documents/CoreOps-Factory/`)
- Third-party dependencies (report upstream)
