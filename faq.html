Frequently Asked

Everything you need to know about Atlas.

What is Atlas?
Atlas is an elite technical platform for builders, operators, and engineers. It combines a powerful CLI (CoreOps), AI-assisted development (Atlas AI), a community marketplace, debate and forum tools, and a competitive benchmarking arena (Gauntlet) — all designed to run natively on Cloudflare infrastructure. It's a philosophy as much as a platform: precision, intelligence, independence.
How do I install Atlas?
Install via curl: curl -fsSL https://raw.githubusercontent.com/DreadHeadHippy/coreops-starter-kit/main/install.sh | bash

Or via npm: npm install -g coreops-cli

Once installed, run coreops --help to see all available commands. Requires Node.js 18+ or a compatible Cloudflare Workers environment.
What is the Gauntlet?
The Gauntlet is Atlas's live competitive benchmarking arena. Tools and approaches are matched head-to-head — for example, CoreOps CLI vs a legacy shell approach — and scored across real metrics like speed, resource usage, and output quality. Community members spectate, vote, and discuss each matchup in real time. Season results are archived for reference.
What is Atlas AI?
Atlas AI is the platform's intelligent build assistant, accessible at /api/atlas and through the interactive chat interface at atlas.html. It's trained on Cloudflare architecture, CoreOps tooling, and operator workflows. Ask it to generate Workers code, debug configurations, explain concepts, or review your infrastructure setup. It's AI-native — not bolted on.
Is Atlas free?
The Atlas core platform is MIT licensed and completely free to use, fork, and deploy — including commercial projects. Premium tools and hosted services may be offered in the future, but the foundational platform, CLI, and community features will always remain open source and free.
Does Atlas work on Termux?
Yes. Atlas and CoreOps CLI are designed with mobile operators in mind. Termux on Android provides a fully compatible environment. Install Node.js via Termux's package manager (pkg install nodejs), then install CoreOps via npm. All CLI features work on ARM and x86 architectures.
How does Cloudflare integration work?
Atlas is built Cloudflare-native from the ground up. It integrates directly with Cloudflare Workers (via Wrangler), R2 object storage, KV namespaces, D1 databases, and the Cloudflare API. Deploy Workers with wrangler deploy, manage secrets via the dashboard or CLI, and use Atlas's built-in audit tools to inspect your Cloudflare configuration. No adapter layers, no wrappers.
What is the Bot Factory?
The Bot Factory is Atlas's four-bot parallel orchestration system:

Builder — generates and deploys Cloudflare Workers from templates.
Watchdog — monitors uptime, latency, and health of deployed services.
Supervisor — coordinates bot execution, handles retries and escalations.
Control Panel — provides a unified interface to manage all bots from a single dashboard.
Can I use Atlas commercially?
Yes. Atlas is MIT licensed, which explicitly permits commercial use, modification, distribution, and sublicensing. You can build products on top of Atlas, fork it for internal tooling, or deploy it as part of a commercial service — with no royalties, no registration, and no restrictions beyond preserving the MIT license notice.
How do I contribute?
Fork the repository at github.com/DreadHeadHippy/coreops-starter-kit, create a feature branch, and open a pull request. Read the CONTRIBUTING guide in the repo root for code style, commit format, and review expectations. Bug reports, documentation improvements, and new module proposals are all welcome via GitHub Issues.
What makes Atlas different?
Three things: Precision — every feature is deliberate, nothing is added for the sake of a feature list. Intelligence — AI is woven into the core, not retrofitted. Independence — MIT licensed with no telemetry, no vendor lock-in, and no platform dependency beyond what you choose. Atlas is the tool that works the way you think, not the other way around.
What's on the roadmap?
The current roadmap includes:

Atlas v2 CLI — plugin architecture, improved audit engine, and new output formats.
Atlas AI v2 — multi-turn context, code execution sandbox, and deeper Cloudflare API knowledge.
Gauntlet Season 2 — expanded matchup categories, persistent leaderboards, and community-submitted challenges.
Market v2 — verified tool listings, ratings, and integrated deployment flows.
