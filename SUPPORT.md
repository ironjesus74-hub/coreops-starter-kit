## Getting Help

CoreOps is a solo-maintained project. Support is best-effort.

**For bugs or unexpected behavior:** [Open a bug report](https://github.com/ironjesus74-hub/coreops-starter-kit/issues/new?template=bug_report.md)

**For feature ideas:** [Open a feature request](https://github.com/ironjesus74-hub/coreops-starter-kit/issues/new?template=feature_request.md)

**For security issues:** See [SECURITY.md](SECURITY.md) — do not open a public issue.

## Scope

This repo covers the CoreOps CLI toolkit and Cloudflare Worker backend.  
For issues with Termux itself, see the [Termux Wiki](https://wiki.termux.com).
